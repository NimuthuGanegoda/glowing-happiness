# Elite Travel Rentals - Audi Q2 Website

A professional travel agency website showcasing the Audi Q2 luxury compact SUV for rent.

## Features

- **Modern Design**: Clean, professional layout with responsive design for all devices
- **Vehicle Showcase**: Detailed information about the Audi Q2 including specifications and premium features
- **Flexible Pricing**: Three rental plans (Daily, Weekly, Monthly) with transparent pricing
- **Online Booking**: Interactive booking form with validation
- **Contact Information**: Easy access to contact details and business hours

## Getting Started

Simply open `index.html` in a web browser to view the website.

Alternatively, you can run a local server:

```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080` in your browser.

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)

## Website Sections

1. **Hero Section**: Eye-catching banner with call-to-action

3. **Pricing Plans**: Transparent pricing with multiple rental options
4. **Booking Form**: User-friendly reservation system
